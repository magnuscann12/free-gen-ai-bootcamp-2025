### Sentence Structure

* **Part 1 (Statement):** `[Time]` + `[Subject]` + `[Adverb/Modifier]` + `[Verb]`
* **Part 2 (Question):** `[Subject]` + `[Verb]` + `[Object]` + `[Completion/Question Structure]`

---

### Clues and Considerations

* To describe *how* the action is being performed (barking loudly), Chinese often uses a structural particle after the verb to connect it to the description, or uses an intensive modifier before the verb. For a simple description of an ongoing or completed state today, you can place a modifier like "very" or "loudly" right before the verb.
* The second sentence asks if an action has already occurred ("Has he eaten?"). To show that an action is completed or has happened in the past, remember to use a completion particle.
* There are a couple of ways to turn the second part into a question. You can either use a question particle at the very end of the sentence, or you can use a "verb + completion particle + negative" structure (asking "eat-completed-not yet?") to ask if it has happened.
